# Programming Notes & Quiz System

A full-stack web application where students can learn programming languages, attempt quizzes, download notes and admin can manage users and content.

---

## Features

- User Login / Registration
- Programming Notes (C, C++, etc.)
- Online Quiz System
- Feedback System
- Password Change
- Admin Dashboard
- Upload Notes (Admin)
- User Activity Tracking
- Database Integration (MySQL)

---

## Tech Stack

Frontend:

- HTML
- CSS
- JavaScript

Backend:

- PHP

Database:

- MySQL

## Installation

1. Download project
2. Move folder to htdocs (XAMPP)
3. Import database:

Open phpMyAdmin → Import → DB.sql

4. Run:
   http://localhost/miniProject/home.php
